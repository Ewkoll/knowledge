GO精确浮点数计算

2020-05-12

浮点数问题在所有的语言开发过程中都会遇到，一旦涉及金融计算等一些需要准确结果的时候，需要重视并解决问题。

Go与其他很多语言（C、C++、Python）一样，使用了IEEE-754标准存储浮点数。

浮点数基础

符号位：1 为 负数，0 为正数。

指数位：存储指数加上偏移量。

小数位：存储系数的小数位的准确或者最接近的值。

32浮点数存储 = 符号位[0:1] + 指数位[1:9] + 小数位[9:32] -> 偏移量(2^(8-1)-1=127)

64浮点数存储 = 符号位[0:1] + 指数位[1:12] + 小数位[12:64] -> 偏移量(2^(11-1)-1=1023)

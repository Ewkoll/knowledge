Spring Jpa 自动创建表大小写规则

默认使用SpringPhysicalNamingStrategy实现, 会把代码中的表定义全部转换成小写.

配置为PhysicalNamingStrategyStandardImpl将不转换任何命名.

implicit-strategy 当不指定命名时候的隐式转换规则.

Jpa中引用关系使用Set会计算Hash会导致数据反复引用.

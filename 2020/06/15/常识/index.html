Ubuntu安装Mysql

1、sudo apt install mysql-server
2、mysql_secure_installation
3、密码 Qa123456.0
4、mysql -u root -p
5、ALTER USER ‘root’@’%’ IDENTIFIED WITH mysql_native_password BY ‘Qa123456.0’;
6、SELECT user, authentication_string, plugin, host FROM mysql.user;
7、select user,host from mysql.user;
8、systemctl restart mysql
9、grant select,insert,update,delete,create,drop privileges on database.* to user@localhost
10、grant all privileges on chain_storage.* to chain_mysql_accout;
11、CREATE SCHEMA chain_storage DEFAULT CHARACTER SET utf8mb4 ;

Redis

1、sudo apt-get install build-essential
2、sudo apt-get install make
3、#include <jemalloc/jemalloc.h>
4、sudo apt-get install autoconf
5、wget https://github.com/jemalloc/jemalloc/releases/download/5.2.1/jemalloc-5.2.1.tar.bz2
6、./autogen.sh –with-jemalloc-prefix=je_
7、make install
8、./deps/hiredis/libhiredis.a: No such file or directory
9、You need tcl 8.5 or newer in order to run the Redis test
10、sudo apt-get install tcl
11、make lua hiredis linenoise
